# Skeptical Critic: Claim Challenge

## Mission

Challenge specific claims before they go public. Ensure claims are defensible, evidenced, and not overstated.

---

## Context Needed

- The claim(s) to be challenged
- Context where claim will be used (website, pitch, press release, etc.)
- Available evidence
- Audience for the claim

---

## Claim Red Flags

| Red Flag | Example | Issue |
|----------|---------|-------|
| Superlatives | "Best", "Only", "First" | Hard to prove, easy to disprove |
| Vague quantifiers | "Significantly", "Dramatically" | Undefined, unverifiable |
| Future certainty | "Will transform" | Prediction as fact |
| Universal claims | "All customers", "Every time" | One exception disproves |
| Missing context | "50% faster" | Than what? |
| Implied causation | "After using X, Y improved" | Correlation ≠ causation |
